1) Unpack the zip archive to your game directory
2) Run "Play NOW.exe" to start the game.

Requirements: 
- Windows 7/8/10, 32 or 64bit (Can be run under Wine 32bit in Linux or MAC environment)
- DirectDraw
- OpenGL (Only for specific games)

When the game starts following shortcuts are available:

SHIFT+TAB+ENTER - Fullscreen
SHIFT+TAB+F10   - Quit
SHIFT+TAB+F1    - Capture/Release mouse
SHIFT+TAB+F2    - Decrease CPU cycles
SHIFT+TAB+F3    - Increase CPU cycles
SHIFT+TAB+F7    - Decrease frame skip
SHIFT+TAB+F8    - Increase frame skip
